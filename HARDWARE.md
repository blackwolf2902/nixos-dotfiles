# Hardware Configuration

## Overview

This repository includes your actual `hardware-configuration.nix` file. This is the hardware-specific configuration generated by NixOS for your system.

## Building

Simply build without any special flags:

```bash
# Build the configuration
sudo nixos-rebuild build --flake .#zenxtsu

# Apply the configuration
sudo nixos-rebuild switch --flake .#zenxtsu
```

No `--impure` flag needed since everything is self-contained in the repo!

## Updating Hardware Configuration

If you change hardware or reinstall, regenerate the hardware configuration:

```bash
# Generate new hardware config
sudo nixos-generate-config --show-hardware-config > /home/shinobi/Downloads/nixos/hardware-configuration.nix

# Commit the changes
git add hardware-configuration.nix
git commit -m "Update hardware configuration"
```

## What This File Contains

The hardware configuration includes:
- Boot loader settings
- File system mounts and UUIDs
- Kernel modules for your specific hardware
- CPU microcode settings
- Network interfaces
- Swap configuration

This file is unique to your hardware and is now part of your dotfiles for easy deployment.
